# stash-plugins

Collection of plugins by matthewmorrone

## Plugins

### Bulk Performer Search
Clicks all performer Search buttons with delay on the Batch Add Performers page.

### Expand All Sources
Adds a button to expand all plugin package sources on the settings page.

### Code Editor
Replaces custom CSS/JS textareas with a fully featured CodeMirror editor with syntax highlighting.

### Scroll Restore
Restores scroll position (enabled on Settings + Scenes list).

### Scene Edit Default
Ensures the "edit" tab is selected when opening a scene.

### Scene Edit Pinned Fields
Adds pin toggles to scene edit fields and moves pinned fields to the top.

### Scene Card Performers
Shows performers on scene cards and allows adding/removing performers inline.

### Scene Performer Roles
Adds small role icons to performer cards on scene pages; stores neutral per-scene/per-performer roles in your browser.

### Configurable Keybindings
Adds configurable keyboard shortcuts for the video player (includes frame-step via time increments).

### Scene Duplicate Checker Tools
Enhances `/sceneDuplicateChecker` with Confirm/Ignore controls and optional duplicate-group tagging.

### Scene Tagger Performer Query
Autofills Scene Tagger query inputs with concatenated performer names (lowercase).

### Hide Installed Plugins
Hides already-installed plugins from the Available Plugins list; they reappear after uninstall.
